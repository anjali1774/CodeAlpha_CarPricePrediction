# 🚗 CarPrice AI - Used Car Price Prediction

## 📌 Project Overview

CarPrice AI is a Machine Learning project that predicts the selling price of used cars based on features such as showroom price, fuel type, transmission, car age, kilometers driven, and ownership.

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-Learn
- XGBoost
- Pandas
- NumPy
- Plotly
- Matplotlib
- Seaborn

## ▶️ How to Run the Project

### Step 1: Clone the repository

```bash
git clone https://github.com/anjali1774/CodeAlpha_CarPricePrediction.git
```

### Step 2: Open the project folder

```bash
cd CodeAlpha_CarPricePrediction
```

### Step 3: Train the model

```bash
python train_model.py
```

### Step 4: Run the Streamlit application

```bash
python -m streamlit run app.py
```
